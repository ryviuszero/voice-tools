# Code of Conduct

Voice Tools Directory is a community project. We want contribution and review to stay respectful, practical, and welcoming.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially in data corrections.
- Keep feedback focused on the work.
- Credit sources and contributors where appropriate.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated off-topic disruption.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block participants whose behavior makes the project unsafe or unproductive.

If you need to report a conduct issue, contact the maintainers through the repository's GitHub moderation tools.
